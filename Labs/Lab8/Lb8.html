CSS stands for Cascading Style Sheets. CSS is a language used to describe how HTML elements should be styled. CSS selections are the part of a CSS rule-set that defines which HTML elements the style should be applied to. The 3 main selectors are name, ID, and class. Name, ID, and class are the elements most important in CSS.
