InLine CSS

An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element. In order to create an inline CSS you must put the CSS into an HTML file instead of an external CSS. Therefore, inline CSS allows the application of a unique style to one HTML element. The usuage is limited but is very beneficial for creating many unique attributes.

Internal CSS

An internal CSS is defined in the head section of an HTML page. However, it is within a style element. The internal CSS is used to define a single HTML page. In order to create an internal CSS file in HTML you must add CSS code in the head section of the document of your choice. The styling changes in internal CSS apply to every specific element found in the file.

External ESS

An external style sheet is used to define the styles for mulitple HTML pages. An external style sheet is accessed through a separate CSS file by creating a link within the head section of the webpage. With external style sheet you are able to create multiple webpages by using the same link to access the stylesheet. The link to an external style sheet is placed within the head section of the page.
